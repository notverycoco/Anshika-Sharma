In this project, I learned the basics of backend development using Node.js and Express.js.

### Features

* View all students
* View a student by ID
* Add a new student
* Delete a student
* Input validation
* JSON data handling

### API Routes

GET /students

GET /students/:id

POST /students

DELETE /students/:id

### What I Learned

* Express.js basics
* API creation
* GET and POST requests
* Route parameters
* Request and Response handling
* Status codes
* CRUD operations
